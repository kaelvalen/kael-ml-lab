# Data Directory

Veri setleri bu klasörde saklanır.

## Yapı

```
data/
├── raw/              # Ham veri setleri
├── processed/        # İşlenmiş veri setleri
└── external/         # Harici kaynaklar
```

## .gitignore

Büyük veri dosyaları Git'e eklenmez. Sadece `.gitkeep` dosyaları ile dizin yapısı korunur.
